# Sotechxy

Public storefront website for **Sotechxy** — no-code automation templates
(n8n, Make, Airtable, GCP, and more) for non-technical people.

**Tagline:** Automations that actually run.
**Live site:** https://mlauradeveloper.github.io/sotechxy/ (custom domain
sotechxy.com connecting)

## Repo structure

```
sotechxy/
├── index.html   ← the website (GitHub Pages serves this)
├── CNAME        ← custom domain config (do not delete)
├── README.md    ← you are here
└── docs/        ← all project documentation
    ├── PROJECT_BRIEF.md    ← source of truth: brand, rules, current state
    └── LAUNCH_ROADMAP.md   ← idea → market-ready launch stages
```

## Working on this project

Start every session by reading `docs/PROJECT_BRIEF.md`. It holds the brand
identity, build principles, data/hosting decisions, and how to work on the
site step by step.
